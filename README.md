# Petite bibliothèque en Go

Ce projet est un mini serveur web permettant d'afficher une liste de livres
ainsi qu'une page détaillée par livre.

## Installation

go mod init main  
go run main.go

## Routes

/         → page d'accueil  
/books    → liste des livres  
/books/{id} → page d'un livre
